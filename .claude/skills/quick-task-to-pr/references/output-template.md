# quick-task-to-pr output artifacts

## `task-brief.md`
- Original requirement
- Context
- Scope
- Out of scope
- Behavioral AC
- Failure modes
- Test strategy
- Key files
- AI complexity estimate

## `e2e-definitions.md`
- One scenario per AC when practical
- Happy path + important unhappy paths
- Explicit expected outcome per scenario
- Note whether the scenario is runnable now or definition-only

## `pr-summary.md`
- What changed
- Why it changed
- Verification performed
- Known limitations / skipped steps

## `review-notes.md`
- Review round number
- New bugs found
- Design concerns found
- Fixes applied
- Remaining blockers
